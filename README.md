# Program:
-- Van 3 láda (arany, ezüst, bronz)-> az egyik egy kincset rejt -> három felirat, melyből csak az egyik igaz --
1. Én rejtem a kincset (arany láda)
2. Nem én rejtem (ezüst láda)
3. Az arany láda hazudik (bronz láda)
